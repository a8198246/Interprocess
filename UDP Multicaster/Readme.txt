описание:
	Тестовое приложение для рассылки рандомных multicast udp пакетов.
	Работает пока не нажать Ctrl+C

зависимости:
	boost 1.56 (или новее), только заголовки, подразумевается расположение в "$(SolutionDir)..\boost\"
	visual leak detector, линковка - динамическая, подразумевается расположение в "$(SolutionDir)..\Visual Leak Detector\"
