описание:
	Тестовое приложение для проверки работоспособности библиотеки interprocess.
	Принимает один аргумент - application id в диапазоне 0 - 12 и эмулирует работу ведомого приложения с соответствующим application id.
	Если аргументов нет, то работает в режиме ведущего приложения.

зависимости:
	boost 1.57 (или новее), линковка - статическая, подразумевается расположение в "$(SolutionDir)..\boost\" (библиотеки в папках boost\stage32 и boost\stage64)
	visual leak detector, линковка - динамическая, подразумевается расположение в "$(SolutionDir)..\Visual Leak Detector\"
